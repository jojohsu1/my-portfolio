Multi-day Weather Forecast App

簡介

1. 地點查詢功能。
2. 當日即時氣溫、文字簡述，氣候、風速、溼度、降雨機率等圖示。
3. 五日內氣候圖示及最高溫、最低溫天氣預報。
4. 攝氏、華氏溫度切換。
5. 日期為GMT標準時間+8時區(台灣時間)。

製作重點

1. 串接API(Google Geocoding API & Dark Sky API)。
2. Webpack。
3. Async、Await / CORS。
4. ES6模組樣式，溫度單位切換。
5. CSS偽元素、動畫，及SASS練習。

Vinyl-Records Website

簡介

1. 簡易購物車功能。
2. 新專輯試聽。
3. 音樂講座。
4. 五種音樂類型網頁。
5. 音樂與生活選物(即將推出)。

製作重點

1. 購物車內商品增加、刪減，數量變更後，總金額異動，無後台功能。
2. 音樂播放器(播放、暫停、更換音源、進度條)。
3. 倒數計時器( 日/ 時/ 分/ 秒)。
4. 揭示模組模式(Revealing Module)。
5. 圖片輪播、下拉式選單、頁籤(Tab)互動切換內容。
